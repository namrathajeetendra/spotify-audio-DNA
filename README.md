# 🎵 Spotify Audio DNA - What Makes Songs Hit Different?

An interactive music analytics app that decodes the science behind popular music, analyzing 98,000+ Spotify tracks across 114 genres.

## 🔗 Live Demo
👉 [spotify-audio-dna.streamlit.app](https://spotify-audio-dna.streamlit.app)

## 📌 Features
- **Genre Audio DNA** — radar charts showing the unique audio fingerprint of each genre
- **The Hit Formula** — compare audio features of hits vs flops side by side
- **Genre Battle** — head-to-head audio profile comparison of any two genres
- **Popularity Predictor** — correlation heatmap revealing which features actually drive popularity
- **Top Artists by Genre** — most popular artists filtered by genre

## 🛠️ Tech Stack
- Python, Pandas, NumPy, Plotly, Streamlit

## 📁 Dataset
[Spotify Tracks Dataset](https://www.kaggle.com/datasets/maharshipandya/-spotify-tracks-dataset) — 98K tracks with audio features including danceability, energy, valence, acousticness, and more
